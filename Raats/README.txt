There are 5 Jupyter notebooks in this repository.

CyTargetLinker_WP = linking linksets to a wikipathway imported as a network.
EXPRandSIGviz = log2FC expression and significance visualization on a pathway that is extended with a dataset
Delete_Nodes = delete nodes that do not have log2FC values. delete nodes that are unwanted. 
EnrichmentScore = calculate enrichment scores and pvalue (hypergeometric distribution).
IndivPath  = add expression and significance data to a pathway. 

It is recommended to walk through the jupyter notebooks in the order stated above. 

Supplementary files can be found in the repository, such as the data file, data processing files and the linksets.

ORA = Raw read count intensity expression calculations
Parsing = Raw read count inensity pre-processing
